# Flux Workflow - 产品文档

## 产品概述

Flux Workflow 是一个**基于 flux-agent 框架的可视化工作流编排工具**，提供前端可视化编排界面 + 后端管理服务。

**核心定位**: 本项目是 flux-agent 的可视化封装层，所有工作流执行、Agent 运行、工具调用等核心能力均依赖 flux-agent 框架提供。

**⚠️ 重要提示**: 任何功能迭代、bug 修复、性能优化都需要仔细参考 flux-agent 的文档和源码，确保与底层框架的行为一致。

## 核心问题

帮助用户通过可视化方式编排 AI 工作流，将 LLM、工具、知识库等组件组合成自动化流程。

## 目标用户

- AI 应用开发者
- 需要构建自动化工作流的技术团队
- 希望低代码方式编排 AI 流程的用户

## 主要功能模块

1. **工作流编排**
   - 可视化拖拽式工作流编辑器
   - 支持多种节点类型（LLM、工具、知识库检索、输入/输出等）
   - 工作流版本管理（草稿、发布、下线）

2. **执行与追踪**
   - 实时执行状态流（SSE）
   - 详细的执行追踪（Trace）
   - 执行历史记录

3. **工具管理**
   - 内置工具库
   - 自定义 Python 代码工具

4. **知识库（RAG）**
   - 向量知识库管理
   - 文档检索节点

5. **Agent 管理**
   - ReAct 模式 Agent 配置
   - Agent 对话界面

## 业务目标

- 提供直观的工作流编排体验
- 支持复杂的 AI 流程自动化
- 降低 AI 应用开发门槛

## 成功指标

- 工作流创建和执行的成功率
- 用户编排效率
- 系统稳定性
